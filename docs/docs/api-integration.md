# API Configuration

## **CoinGecko API Integration**
We fetch live cryptocurrency prices from CoinGecko:

*API Endpoint*:  
   - https://api.coingecko.com/api/v3/coins/markets

Parameters:  
    - *vs_currency* = usd  
    - *ids* = bitcoin,ethereum,ripple,cardano,solana
